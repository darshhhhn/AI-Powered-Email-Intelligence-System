# AI-Powered Email Intelligence System

An AI-powered email management platform built using Django and Python.

## Features

- User Authentication
- Gmail IMAP Integration
- Email Retrieval
- Email Database Storage
- Intelligent Email Search
- AI-powered Email Thread Summarization
- Timeline Extraction
- Recommendation Generation
- Professional PDF Report Generation

## Technologies

- Python
- Django
- SQLite
- IMAP
- LLM Integration
- ReportLab

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
